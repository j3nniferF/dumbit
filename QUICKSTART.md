## ğŸ› Troubleshooting

**Problem: Can't drag tasks**
- Solution: Make sure you're clicking on the task row, not the checkbox

**Problem: Edit doesn't save**
- Solution: Press Enter or click outside. ESC cancels.

**Problem: Import doesn't work**
- Solution: Make sure you're using a file exported from this app

**Problem: Clear Completed button missing**
- Solution: Refresh the page. Make sure JavaScript is enabled.
